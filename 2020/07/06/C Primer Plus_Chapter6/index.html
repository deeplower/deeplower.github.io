6.1.1 程序注释

对于这一个循环语句，只有循环体最后一句的东西需要讨论。

根据我们对这一语句的分析，这一语句是判断语句，其作用是输入一个数字的时候转变为1，输入一个字母的时候转变为不是1 的数字。

书的构思：

目的：

  1. 每次循环都要获得一个num 的值（输入字母结束）；
  2. 每次输入数字的时候都要重置status；

实现方法：

利用scanf() 的两个特性来完成。

  1. scanf() 就是用来读取数值的大小的嘛；
  2. scanf() 返回成功读取项的数量—简单理解就是scanf()语句的值为读取成功的项的数量（格式正确就读取了）；

这样的话就成功地利用一条语句更新了两个变量的值—> num, status.

这样就导致了这样一个结果：对于数字的变量成功存储到num 中，不是数字的变量不能成功存储，于是scanf() 语句的值就为0，这样当用户结束的时候，引导用户输入不是数字的q就行了。

注：我不懂的地方：这里作者提到了一个特殊情况：如果scanf() 在转换值之前出了问题（检测到文件结尾或遇到硬件问题），会返回一个特殊值EOF（其值通常被定义为-1），这个值也会使循环终止。—为什么会这样呢？我不晓得。

比较其他解决方案

假如scanf() 没有返回值，那么：

1.用num 的值来判断是否终止循环—>会导致有一些值可能不能参与求和，直接就结束了；

2.在每一次循环之后放置一个判断语句，用户通过输入来决定是否再次执行循环体或者输出求和结果—>减慢了输入速度，不流畅；

总而言之，不如利用scanf() 的返回值来得顺畅。

总结该程序的结构

1.把sum 的值初始化为0；

2.提示用户输入数据；

3.读取数据；

4.当输入的数据是整数的时候；

  • 输入添加给sum；
  • 提示用户输入；
  • 读取下一个输入；

5.输入完成后，打印sum 的值。

给一串人能读懂的文字起名叫做伪代码(pseudocode)，就是在构思代码时期写的文字，与计算机语言相对应。初学者一定要多练习这个过程（可以不用考虑代码的细节、语法和代码实现）。

2.当测试为真时；

6.2 While 语句

while 循环语句的通用形式：

statement 部分可以是一句语句，也可以是多句语句；区别是，一条语句不用花括号，多条语句要用花括号。

expression—任何关系表达式—值之间的比较

如果 expression 为真（更一般地说：非零），执行 statement 一次；然后再次判断 expression。直至 expression 为假（0）为止。

注：每一次循环都称为一次迭代（iteration）。

6.2.3 while：入口条件循环

while 循环是使用入口条件的有条件循环。所谓“有条件”值得就是语句部分的执行取决于测试表达式描述的条件，如（index < 5)。该表达式是一个入口条件（entry condition），因为必须满足条件才能进入循环体。在有的情况中，就不会进入循环体，因为为条件一开始就为假。

6.2.4 语法要点

  • statement 部分可以是一句语句，也可以是多句语句；区别是，一条语句不用花括号，多条语句要用花括号。
  • while 循环while（statement）后面不能带分号，不然就相当于循环体就是一个单独的分号，空语句。

6.11 数组简介

数组：array

  • 一个错误：将不存在的数组元素赋值或者初始化；

6.11.1 在for循环中使用数组

for循环在数组中的应用：遍历数组。

遍历数组的作用：

  • 定义数组（输入语句在循环体中）；
  • 输出数组（建议输入之后立马输出）；
  • 将数组的所有元素或者部分元素求和或进行处理；

相关的编程风格以及注意事项：

1、用#define指令创建明确的明示常量：就是用SIZE作为常量来定义数组的大小，以便后续修改，不用在修改的时候注意每一个细节；

3、很好的编程习惯：程序能重复显示刚读入的数据——确保程序处理的数据与期望相符。

4、是使用独立的循环还是合并的循环（for loop）

  • 根据情况。
  1. 合并的for loop 能够使得程序更加紧凑；
  2. 独立的for loop 可能更加模块化。
  • 什么是模块化？
  1. 程序中独立的单元，每个单元完成一个任务。
  • 模块化有什么好处？
  1. 增加程序的可读性；
  2. 因为层次分明，更加便于修改、更新，可以把成块的代码放进函数里。

6.13 关键概念

现在是愉快抄书环节

  • 创建循环：
  1. 测试条件（最后的判断）确保是有限循环；
  2. 确保循环测试中的值在首次使用之前就被初始化了；
  3. 确保每次测试的值要更新（和第一点遥相呼应）；
  • 使用函数的三步：
  1. 通过函数原型声明函数；
  2. 在程序中调用函数；
  3. 定义函数；

现代的编程习惯是吧程序要素分为结口部分和实现部分，更有利于程序员的思考，和程序框架的搭建。
