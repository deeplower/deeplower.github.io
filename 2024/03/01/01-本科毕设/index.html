作者：张竞，物理与电子信息学院

摘要：低照度图像由于亮度低、明暗不均匀，导致图像信息少，可读性差，因此低照度图像增强是图像处理的一项关键技术。同态滤波是图像增强的算法，但是其传递函数中所包含的参数较多，在陌生的问题场景时，只能采用较为繁琐的实验方法找出较为合适的参数。因此，本文针对生活中常见的低照度问题，采用照度反射模型的同态滤波方法进行图像增强，并将同态滤波的传递函数的多个参数转变成单参数，使得在面对不同问题场景时，可以通过更加简洁的方法快速找到合适的参数。

关键词：同态滤波；增强算法；控制变量法；参数分析

Homomorphic Filter-based Gray Image Enhancement Algorithm

Jing Zhang, School of Physics and Electronic Information

Abstract: Low illuminance image enhancement is a key technology in image processing due to its low brightness and uneven brightness, resulting in limited image information and poor readability. Homomorphic filtering is an algorithm for image enhancement, but its transfer function contains many parameters. In unfamiliar problem scenarios, cumbersome experimental methods can only be used to find more suitable parameters. Therefore, this article focuses on the common low illuminance problem in daily life and adopts the homomorphic filtering method of illuminance reflection model for image enhancement. It transforms multiple parameters of the homomorphic filtering transfer function into single parameters, so that when facing different problem scenarios, suitable parameters can be quickly found through more concise methods.

Key words: Homomorphic filter; enhancement algorithm; controlled variable method; parameter analysis.

1 绪 论

1.1 研究背景

图像增强是数字图像处理中的一项重要技术，其主要目的对各种图像进行处理，以便于人或者计算机识别出更多、更有用的信息。在实际应用中，图像往往受到外部环境、信息传输等等条件的影响，如光照不均、噪声干扰、图像压缩等，导致图像质量下降，难以满足后续处理和分析的要求。因此，如何有效地增强图像质量成为了数字图像处理领域中的重要的研究部分。

在图像增强中，同态滤波作为一种重要频域方法，具有许多优点，如使用符合光学成像原理的照度反射模型对图像进行建模，对光照不均匀性和噪声具有很好的适应性；同时对不同种类的图像进行图像增强也具有普适性和鲁棒性等。因此，同态滤波在图像增强领域得到广泛的研究和应用。

1.2 国内外研究现状

随着各项技术的不断发展，人们对图像处理技术解决现实问题的需求越来越迫切[1][2]。从20世纪40年代计算机技术进入数字图像处理领域以来，其技术的更迭就未曾停止[3]。图像增强算法就是其中的一个重要分支。人们利用图像增强算法解决过很多现实中常见的问题[4][5]。

图像增强的方法大致可以分为空间域和频域两大类，其中每一种方法都其专门的应用领域。同态滤波方法属于频域方法中的一种适用性较广的方法[6]。同态滤波创新性地将频域方法和灰度变换结合[7]。在处理动态范围很大，细节难以分辨的图像时，同样需要用到同态滤波方法[8]。同态滤波是基于照度反射模型的增强方法，它将元灰度值看作是代表低频成分的照度和代表高频成分的反射率两个组成部分，是一种更加灵活的频域处理方法[9]。

国内在同态滤波的图像增强方面拥有很多的研究成果：2016年，西安邮电大学程新[10]基于低照度图像，分析了同态滤波方法中不同的滤波器作用于模型之后对低照度图像增强结果的不同影响；2021年，上海理工大学洪琛等[11]同样针对低照度图像，在同态滤波中使用高斯滤波器，并细致分析了不同参数对结果的影响。两者都是针对低照度图像，然后分析了不同的滤波器类型以及滤波器的参数对实验结果的影响。2020年，重庆邮电大学冯相辉[12]采用了一种改进的同态滤波图像增强算法，使得参数分析的实验设计更为便捷。本文将在前人的基础上，更加细致地分析各个滤波器类型的参数对最终实验结果的影响。

随着各个领域对图像增强的要求越来越高，图像增强的研究也拥有了更重要的价值。因此，不断深入研究图像增强技术也显得越来越重要。

1.3 研究方法和论文结构安排

2 图像增强的基本理论和方法

在数字图像处理领域，有着多种多样的图像增强算法。每一种图像增强算法都有其适用的领域。为了更好地理解同态滤波在处理光线照度不均、薄雾等环境问题时的优越性，本章介绍几种常见的图像增强技术。图像增强技术主要分为空间域和频域两类，其中空间域增强直接对图像进行处理，是较为基本、常用的一类图像增强方法。空间域方法中具有代表性的有灰度变换和直方图修正等。频域方法则利用正交变换将图像从空间域转换到频域，在频域中进行增强处理，然后通过反变换将图像恢复到空间域，得到最后增强的结果。频域方法中常见的有低通滤波、高通滤波、同态滤波等。本章将介绍几种常见的空间域图像增强和频域图像增强的基本理论和方法，以及各种图像增强算法的对比。同态滤波的基本理论将在下一章进行介绍。

2.1 空间域图像增强

​ 一幅图像可以表示为一个二维函数，由位置信息获取该点的像素值。如果对原始图像的像素点直接进行线性变换和非线性变换，则该方法称为空间域的图像增强算法。

空间域增强有两种常见的算法：点运算和邻域运算。下面分别对这两种运算进行介绍。

2.1.1 点运算

点运算是指在图像空间中直接进行像素级的运算，即对每一个像素进行独立的运算。相比于邻域变换而言，点运算的变换结果完全由该点的像素值本身决定，和其邻域无关。常见的点运算有灰度变换和直方图修正两种。

2.1.1.1 灰度变换

灰度变换是一种简单的空间域图像增强方法，它通过对像素的灰度值进行改变，来调整图像的对比度和亮度，从而增强图像的视觉效果。

2.1.1.2 直方图修正

直方图修正分为直方图均衡化和直方图规定化两种。其中直方图修正化有明确的目的：改善对比度，增强视觉效果；而直方图规定化是根据给定的参考图像对待处理图像进行处理。

  1. 直方图均衡化

直方图均衡化属于直方图修正，是一种常见的图像增强技术。该技术的核心思想是重新分配图像中像素的灰度级，使得灰度级分布更加均匀，从而增强图像的对比度。直方图均衡化的过程如下：

​ （1）统计图像中各个灰度级的像素数量，生成灰度级分布直方图。

​ （2）对于每个灰度级，计算其在灰度级分布直方图中的累积概率分布。

​ （3）对于每个像素，将其灰度级替换为对应的累积概率分布值，并将结果映射到0-255的灰度级范围内。

​ （4）生成经过均衡化后的直方图，该直方图的灰度级分布更加平均，从而提高了图像的对比度和视觉效果。

直方图均衡化可以应用于灰度图像和彩色图像，常见的算法包括全局直方图均衡化、局部直方图均衡化等。直方图均衡化在图像处理、计算机视觉等领域具有广泛的应用，如图像增强、图像分割、目标检测等。

  1. 直方图规定化

直方图规定化也称为直方图匹配，是一种基于参考图像的图像增强技术，用于将待处理图像的灰度级分布调整为参考图像的灰度级分布，从而达到与参考图像相似的视觉效果。该方法和直方图均衡化的区别是：直方图均衡化的目的非常明显，是为了改善图像的对比度；而直方图规定化是根据给定的参考图像对待处理图像进行处理。

直方图规定化可以应用于灰度图像和彩色图像，常见的算法包括全局直方图规定化、局部直方图规定化等。直方图规定化在图像处理、计算机视觉等领域具有广泛的应用，如图像增强、图像配准、目标跟踪等。

直方图修正的两种方法都有其重要应用：直方图均衡化在针对对比度较低的图像非常有效，例如医学CT成像；但是由于直方图均衡化是全局性的改变，可能会导致图像区域的某些细节丢失；以及在对于存在异常值的图像，会产生不理想的结果，例如：含大量椒盐噪声的图像。相比之下，直方图规定化更具灵活性，可以调整局部区域的图像。但是直方图规定化需要设置参考图像，步骤较为复杂。

直方图修正属于点运算的方法，没有考虑到像素点之间的关系，在面对突出边缘、模糊图像等需求时效果不佳。因此需要其他图像增强算法，即为下一节介绍的邻域运算。

2.1.2 邻域运算

​ 邻域运算是图像增强中重要的技术手段之一，能够对图像进行局部处理，以实现去噪、增强边缘等目的。

2.1.2.1 平滑滤波

平滑滤波是一种常用的图像处理技术，也被称为模糊滤波。其作用是通过对图像像素进行加权平均来减少图像中的高频噪声和细节信息，从而使图像变得更加平滑。平滑滤波可以通过卷积操作实现，其中卷积核包含一组权值，用于计算图像中每个像素的新值，具有模糊图像的作用。

常用的平滑滤波包括均值滤波、高斯滤波等。均值滤波使用一个固定大小的卷积核对图像像素进行平均，因此它可以有效地去除高频噪声，但是也会使得图像细节信息丢失。高斯滤波使用一个高斯分布的卷积核对图像像素进行加权平均，可以在去除高频噪声的同时保留图像的细节信息。

平滑滤波在图像处理中有着广泛的应用，例如在图像去噪、图像模糊等方面都有着重要的作用。然而，需要注意的是，过度的平滑滤波（例如选取的卷积核过大），可能会导致图像失真和信息丢失。因此在使用平滑滤波时需要根据具体应用场景进行权衡和选择。

2.1.2.2 锐化滤波

锐化滤波是一种常用的图像处理技术，它可以增强图像的边缘和细节信息，使图像看起来更加清晰和锐利，常用于图像的边缘检测等方面。与平滑滤波不同的是，锐化滤波使用的卷积核通常是一组带有负权值的系数，用于减少像素值的平均值，从而使得像素之间的差异更加明显。

在实际应用中，锐化滤波可以通过卷积操作来实现，其核心思想是将原始图像与一个锐化卷积核进行卷积运算，以得到增强后的图像。常用的锐化卷积核包括拉普拉斯算子、Sobel算子、Prewitt算子等。

其中，拉普拉斯算子可以通过在中心像素周围取8个邻域像素的值，并对它们进行加权求和来实现锐化效果。Sobel算子和Prewitt算子可以用于边缘检测，并通过对水平和垂直方向的像素进行卷积运算，以得到边缘信息和方向信息。

需要注意的是，锐化滤波通常会增加图像的噪声和伪影，因此在使用锐化滤波时需要根据具体应用场景进行权衡和选择。同时，为了避免锐化效果过度导致的图像失真和信息丢失，通常需要在锐化滤波之前进行平滑处理，以减少噪声和平滑图像。

平滑滤波和锐化滤波都是基于邻域运算的图像增强算法，在不同的邻域有着各自的应用。相较于点运算，邻域运算已经考虑到了像素点与点之间的联系。而频域的图像增强算法更加关注像素点之间的关系，将在下一节介绍。

2.2 频域增强

常用的频域图像增强算法有低通滤波、高通滤波以及同态滤波。下面主要介绍低通滤波和高通滤波，同态滤波将在下一章节进行介绍。

2.2.1 低通滤波

低通滤波是一种线性滤波，它对于输入图像的每个像素都计算一个输出值，这个输出值是该像素及其邻域像素的加权平均值。作为一种常见的图像处理技术，其设计通常基于其频率响应。频率响应是滤波器对于不同频率分量的响应，由滤波器的传递函数决定。低通滤波常用于平滑图像并降低高频噪声，例如去除图像中的颗粒状噪声或纹理。

常见的低通滤波器有均值滤波、高斯滤波、中值滤波等。均值滤波器计算邻域像素的平均值来平滑图像。高斯滤波器是一种加权平均的滤波器，它利用高斯函数来对邻域像素进行加权，从而平滑图像。中值滤波器则通过对邻域像素进行排序，选取中间值来代替中心像素，从而去除图像中的椒盐噪声。

低通滤波不仅仅在图像处理中应用广泛，在各种信息处理的过程中，比如：音频信号、视频信号中都有十分重要的作用。

2.2.2 高通滤波

高通滤波同样是一种常见的图像处理技术，用于增强图像的高频细节信息，如图像中的边缘和纹理。与低通滤波常用于的图像平滑来看，高通滤波可以更多地用于显示图像中边缘特征，因此常用于图像的特征提取。

低通滤波、高通滤波等频域图像处理技术在图像增强领域应用非常广泛，其中高通滤波类似于邻域运算中的锐化滤波，可以提取图像的边缘特征；低通滤波类似于邻域运算中的平滑滤波，可以使图像变得模糊。但是这两种频域图像处理算法在解决卷积性和乘性噪声时效果不好，需要其他算法。因此在下一节介绍频域方法中的同态滤波。

2.3 同态滤波的优势

由上文的描述我们得知：同态滤波在解决由光照、薄雾等环境问题导致的视觉问题时有充分的优势。相对于空域滤波，如果图像在空域上相关性明显，借助于傅里叶变换可使在空域的复杂计算转换到频域后得到简化；相对于其他频域滤波的优势，即：已知大多数频域滤波可以解决加性噪声，而同态滤波解决乘性或卷积性噪声。同态滤波可以使图像处理符合人眼对于光学成像的原理（照度反射模型），如果直接使用一般的频域滤波方法，则可能导致最终的图像产生一定的失真。

同态滤波还具有以下特点：在保证图像原本的特征不丢失的情况下，能够尽量扩大相距较近的像素点的对比度，同时使得图像的整体动态范围不至于太大。

2.4 本章小结

3 传统同态滤波的图像增强算法的研究与实现

总结与展望

本论文研究了同态滤波的图像增强算法，解决低照度场景下的图像增强算法，并在参数分析方面和传递函数的改进对传统同态滤波方法进行补充。主要完成以下工作：

（1） 实现传统的同态滤波的灰度图像增强算法，并了解其他图像增强算法；

（2） 对低照度场景下的同态滤波传递函数的参数进行实验研究；

（3） 改进传统的多参数的同态滤波传递函数，将其设计为单参数的传递函数，更易于实验分析合适的参数。

当今图像增强技术的发展极其迅速，同态滤波算法也越来越趋于成熟，在光线照度不均改善、薄雾去除、图像分类等方面发挥着重要作用。但是随着图像处理的任务越来越复杂化，任务场景更加多样，在实际操作中仍然会遇到各种各样的问题，传统的同态滤波算法也不能保证面面俱到。因此不断进行改进，并提高系统的自适应能力，才是未来的大方向。

对于同态滤波算法，还可以使用卷积神经网络进行更深层的图像增强，可以对图像的各种细节进行处理；或者满足开发者各种不同的需求。定制化、多样化，这同样是是图像增强领域值得研究的方向。
